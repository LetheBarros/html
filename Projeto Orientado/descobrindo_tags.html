Descobrindo Tags

Aqui estão as tags que pesquisamos neste trabalho. Usamos mais tags do que as que descrevemos para a disposição do conteúdo e apresentação das tags para nos atermos ao requisitado.

<nav></nav>

Esta é uma tag que faz a separação semântica de conteúdos/links de navegação entre páginas ou dentro dela mesma. Ela deve envolver as tag <a></a>, por exemplo.

Exemplo de utilização:

    <nav>
        <a href="index.html">Home</a> |
        <a href="leticia_barros.html">Letícia Barros</a > |
        <a href="sabrina_da_silva.html">Sabrina Allexia Galvão da Silva</a> |
        <a href="links_externos.html">Links Externos</a >|
        <a href="tabela_de_curiosidades.html">Tabela de Curiosidades</a > |
        <a>Descobrindo Tags</a>
    </nav>

O resultado é como o apresentado na navegação das nossas páginas.

<footer></footer>

Acompanhando a lógica das tags head e body, a tag footer funciona como uma separação (semântica) para a página, como um rodapé. Geralmente tem conteúdos como autoria, data de publicação, direitos autorais e até alguns links de navegação.

Exemplo de utilização:

    <footer>
        <p> Copyright © 2021 Letícia Barros e Sabrina da Silva</p>
    </footer>

<input>

Produz uma área de texto editável para o usuário. Pode estar presente acompanhando outras tags, como o form de exemplo abaixo.

Possui também atributos vinculáveis a respeito de qual será o conteúdo de entrada esperado, tamanho e título.

Exemplo de utilização:

        <label>Nome: <input></label>

<form></form>

Esta tag apresenta um formulário interagível para o usuário, que pode conter diversas outras tag para espaços para textos, checkbox e botões por exemplo, que posteriormente podem ser enviados para o servidor.

Exemplo de utilização:

    <form>
        <label>Nome: <input></label>
        <fieldset>
            <legend>Gênero</legend>
            <label> <input type=radio name=genero> Feminino </label>
            <label> <input type=radio name=genero> Masculino </label>
        </fieldset>
    </form>
